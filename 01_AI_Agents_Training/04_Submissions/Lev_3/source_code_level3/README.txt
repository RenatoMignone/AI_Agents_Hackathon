Level 3 source package for Reply AI Agents sandbox submission.

Files:
- main.py: wrapper that defaults to --level 3
- sandbox_agent.py: main solver implementation
- requirements.txt: Python dependencies
- .env.example: environment variable template

How to run (example):
python main.py --model meta-llama/llama-3.1-8b-instruct

Expected output file:
outputs/public_lev_3_predictions.txt
